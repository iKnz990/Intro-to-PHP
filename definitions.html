2-1: Download Git Client & Create a Repository

Version Control Software : A system that records changes to files over time, allowing collaboration and tracking of modifications.


Add: Stages specific changes for inclusion in the next commit.


Commit: Creates a snapshot of staged changes, recording them in the repository's history.


Push: Updates a remote repository with commits made in the local repository.


Pull: Fetches and merges changes from a remote repository into the local repository.


Clone: Creates a local copy of an existing remote repository, including all files, history, and branches.
